매쓰홀릭 기술 검토 요청

회의 종료 후, 매쓰홀릭 담당자에게 자사관의 클래스인 기반 인터랙티브 수업을 기술적으로 시연·설명했습니다. 매쓰홀릭 측으로부터 "향후 검토하겠다"는 답을 받았으며, 아래는 현재 운영 도구(클래스인·오르조)의 활용·불편점과 매쓰홀릭에 검토 요청한 사항을 정리한 내부 보고 자료입니다.

01 · 클래스인 (ClassIn)

실시간 인터랙티브 서술형 수업

전자칠판·패드 기반으로 선생님과 학생 풀이가 실시간으로 오가는 수업 도구. 자사관 인터랙티브 수업의 핵심.

클래스인 활용 현황

활용 방식
  • 선생님이 전자칠판으로 서술형 수업 진행
  • 선생님 필기가 학생 화면에도 동시에 표시
  • 학생도 자신의 패드에 풀이 필기 작성
  • 학생 풀이를 분할화면으로 실시간 관찰 + 실시간 코멘트
  • 완성된 학생·선생님 필기가 사진 저장 → 원할 때 복습 가능
불편한 점 → 매쓰홀릭 해결·요청
  • 모든 문제를 매쓰플랫에서 출제 후 캡처·재편집→ 매쓰홀릭 강의모드로 출제·편집 일원화 기대
  • 저장 사진의 선생님↔학생 풀이 매칭이 어려워 별도 홈페이지(클래스인 아카이브) 운영 중
  • 풀이 확인에 손이 많이 감→ ① 학생이 자기 풀이노트 확인 ② 선생님 풀이 저장·제공 ③ 한 문제에 선생님+학생 풀이 동시 비교 (요청·검토)
추가 필요 기능 · 학생 풀이에 선생님이 코멘트를 달아줄 수 있는 기능
02 · 오르조 (Orzo)

문제 제출·풀이 수합 도구

학생이 문제를 풀어 제출하면 풀이가 함께 수합되고, 시간 정보와 학부모 공유까지 제공.

오르조 활용 현황

활용 방식
  • 학생이 문제를 받아 풀고 제출
  • 제출 시 학생 풀이가 함께 선생님께 제출됨
  • 문제 풀은 시간·오답 진행 시간 확인
  • 풀이를 학부모에게 공유하는 기능 제공
불편한 점 → 매쓰홀릭 해결·요청
